We need a way to be able to view the statistics of our servers on major games such as Minecraft, Ark, and Rust without needing
to dig through folder and files, those are times of the past. Today we need a universal server dashboard displaying server statistics
and making it easier on the user to get insights on what their playing and how its afffecting their performance

Key Features

    Live Server Status: Quick-glance indicator to see if the server is currently online or offline.

    Simplified Commands: Dedicated buttons for initializing and stopping server boot sequences without manual terminal interaction.

    Whitelist Management: An integrated input field to easily add players to the server whitelist.

    Responsive Layout: A modern interface built to handle administrative tasks efficiently.

Files Used

    Frontend: HTML, CSS,JavaScript.

    Backend: node.js

How to Get Started

    Clone the repository:
    Bash

    git clone https://github.com/DamienZarich/ServerLoaded

    Install dependencies:
    Bash

    npm install

    Launch the monitor:
    Bash

    Access the dashboard: Open your browser and navigate to http://localhost:3000.

Future Features

    Player Slot Monitoring: Displaying the current number of players versus max slots.

    Real-time Logging: A dedicated area to view server console output directly in the browser.

    Secure Login: Implementing an authentication layer for administrative access.
